# Issues de Sprints de X64DOS
>
> **Cumplimiento:** ISO/IEC 12207, 25010, 15504, 27001, 9001.  
> **Documento versionado y auditado.**  
> **Revisión:** 2026-05-14, **Responsable:** Equipo X64DOS

Este directorio almacena incidencias (bugs, mejoras, tareas) ligadas a los sprints activos.

## Reglas de documentación

- Usa la plantilla en `issue-template.md` para cada incidencia, incrementando el número correlativo.
- Relaciona cada issue con su sprint correspondiente.
- Cruza acciones correctivas con logs, evidencia, scripts y responsables.
- Cierra issues sólo con validación QA/documentación.

## Navegabilidad y referencias

- Desde los documentos de cada sprint enlaza los issues con `[issue-XXX](issues/issue-XXX.md)`.
- Usa títulos de issues claros y alineados al objetivo industria.
